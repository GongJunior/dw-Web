The purpose of this project was to make an easy access, client-side, roll generator for my personal needs. This was heavily inspired from https://www.rempe.us/diceware/, with a few divergences for my own convenience. I encourage you to check out this site as there is a lot of solid information if you'd like to learn more.

Currently, this tool has a couple simple uses. First, the main page can be used to generate a passphrase or a set of rolls. Secondly, if you'd like to fetch the entire dataset in JSON format, you can access {{ url_for('generate.get_data', _external=True) }}. This file contains all loaded lists in an easily parsable format. Lastly, if you'd like run the app locally, this project can be installed from Github. Running it locally will allow you customize your own list set if necessary.
